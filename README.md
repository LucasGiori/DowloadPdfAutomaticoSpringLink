# DowloadPdfAutomaticoSpringLink

Este projeto foi desenvolvido para realizar o download automático de alguns Livros, que estão gratuitos na data 27/04/2020, mais de 400 pdf disponiveís, o algoritmo acessa a pagina e encontra o link do pdf automatico e realiza o dowload.

# Arquivo lista.txt
No arquivo lista.txt é onde vai os link da paginá de apresentação do material (PDF)

# Arquivo scraping_spring_link.py 
Arquivo pyhton que faz o scraping da página e salva o PDF

# Bibliotecas Utilizadas
requests,lxml,bs4 \n
