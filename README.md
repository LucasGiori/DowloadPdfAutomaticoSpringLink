# DowloadPdfAutomaticoSpringLink

Este projeto foi desenvolvido para realizar o download automático de alguns Livros, que estão gratuitos na data 27/04/2020, mais de 400 pdf disponiveís, o algoritmo acessa a pagina e encontra o link do pdf e realiza o dowload.

## Arquivo lista.txt
No arquivo lista.txt é onde vai os link da paginá de apresentação do material (PDF)

## Arquivo scraping_spring_link.py 
Arquivo pyhton que faz o scraping da página e salva o PDF

## Bibliotecas Utilizadas
1. *requests*
2. *lxmL*
3. *bs4*
### Comando para dowload de bibliotecas python através do PIP
#### 1. pip install requests
#### 2. pip install lxml
#### 3. pip install bs4
